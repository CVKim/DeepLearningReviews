# Industrial AI Research Portfolio

Industrial AI / Computer Vision / Vision-Language Model / Predictive Maintenance Research Repository

---

## Overview

This repository organizes research papers, implementation reviews, and technical trends focused on industrial artificial intelligence.

The repository is designed as a long-term research portfolio covering:

- Defect Generation
- Industrial Anomaly Detection
- Vision-Language Models (VLM)
- Predictive Maintenance with LLM + Time-Series

The goal is not only paper summarization, but also understanding how modern AI technologies can be integrated into real industrial systems.
